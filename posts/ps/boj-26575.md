문제 설명이 영어긴 한데 입력 내용을 보면 대충 a _ b _ c를 출력하는 것을 알 수 있다.

```c++
#include <iostream>

using namespace std;

int main() {
    int N;
    cin >> N;

    cout << fixed;
    cout.precision(2);

    for(int i = 0; i < N; i++) {
        double a, b, c;
        cin >> a >> b >> c;
        cout << "$" << a * b * c << '\n';
    }
}
```
